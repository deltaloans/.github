# deltaloans

### Research Into Flash Loans and Interface Designs to Create Value From Debt

> 47 repositories covering flash loan protocols, lending infrastructure, liquidation bots, stablecoin mechanics, and DeFi debt instruments.

---

## What This Is

deltaloans is a **DeFi lending research library** — a focused collection of every major flash loan, lending protocol, and debt instrument smart contract that has been deployed and audited on mainnet. The thesis: debt is not just a liability — it is a composable primitive that creates value when properly structured.
